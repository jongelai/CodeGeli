Aquí tienes algunos enlaces útiles para desarrollar código, por ejemplo el editor de codigo Visual Studio Code, que es el que utilizo siempre, y Firebase, donde podrás alojar tus webs personales

  • Visual Studio Code: Un editor de código gratuito, fácil de usar y personalizable.
  • JavaScript: El lenguaje principal para mis proyectos.
  • Node.js: Entorno de ejecución para JavaScript, necesario para el servidor Firebase.
  • ChatGPT: Una herramienta de inteligencia artificial perfecta para responder preguntas, resolver problemas de código y aprender sobre programación.
  • Firebase: Una plataforma gratuita de Google, ideal para alojar tus webs gratis.
